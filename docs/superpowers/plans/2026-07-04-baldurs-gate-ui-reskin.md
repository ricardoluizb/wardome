# Baldur's Gate-style UI Reskin Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Reskin `web/play.html`/`web/style.css` with a dark stone-and-leather Baldur's Gate look — textured panel backgrounds, gold double-border framing with carved corner accents (heaviest on the room-art portrait), gilded stat bars, and a fantasy serif heading font — without touching `web/client.js` or any gameplay code.

**Architecture:** One new AI-generated texture image (`web/assets/ui/panel-texture.jpg`, via a one-off `tools/gen-ui-texture.js` script following this project's existing `gpt-image-1` pattern), then a pure CSS pass on `web/style.css` plus a `<link>` addition to `web/play.html` for the Google Fonts import. All ornamentation (corner accents, double borders) is done via layered `background-image`/`box-shadow` CSS, not new markup or new pseudo-element wrappers — `background-image` renders on `<img>` elements exactly like any other element, so `#room-art` gets the same layering technique as the panels with zero HTML structure changes.

**Tech Stack:** Plain CSS3 (layered `background-image`, `box-shadow`, `background-blend-mode`), Google Fonts (Cinzel), Node.js + OpenAI `gpt-image-1` API (existing pattern from `tools/gen-world-art.js`/`tools/gen-landing-art.js`).

## Global Constraints

- No changes to `web/client.js` — this is a presentational-only reskin. `client.js`'s existing DOM lookups (`document.getElementById('room-art')`, `'hp-bar-fill'`, etc.) and its inline `style.backgroundColor`/`style.width` writes in `setBar()` must keep working unmodified.
- No changes to the HP/mana/move stat bar *colors* (`#e05252`/`#4a90d9`/`#d4af37`, set inline by `client.js`'s `setBar()`) — only their frame/bevel in CSS.
- No id/class renames on any element `client.js` references (`output`, `room-id`, `room-art`, `hp-bar-fill`, `hp-text`, `mana-bar-fill`, `mana-text`, `move-bar-fill`, `move-text`, `stat-line`, `input-form`, `command-input`).
- `#output` and `#command-input` (and `.stat-label`/`.stat-value`) stay monospace — do not apply the new serif font there (ANSI column alignment and numeric readability depend on a monospace font).
- `OPENAI_API_KEY` is read only from the environment at script runtime (`process.env.OPENAI_API_KEY`) — never hardcoded, committed, or printed. Source it from `~/Documents/WHCreative/apps/web/.env.local` into your shell before running the generation script, the same way used for `tools/gen-landing-art.js`/`tools/gen-world-art.js` earlier in this project.
- No automated test suite for this project (locked decision) — verification is manual/observational: rebuild, reload in a browser, visually confirm.

- [ ] **Step 1: Write `tools/gen-ui-texture.js`**

```js
// tools/gen-ui-texture.js
// One-off: generates the Baldur's Gate-style UI reskin's panel background
// texture via OpenAI's image API (gpt-image-1), following the same pattern
// as tools/gen-landing-art.js and tools/gen-world-art.js.
//
// Requires OPENAI_API_KEY in the environment. Never hardcode the key here.
const fs = require('fs');
const path = require('path');

const OUT_PATH = path.join(__dirname, '..', 'web', 'assets', 'ui', 'panel-texture.jpg');

const PROMPT =
  "Seamless background texture: ancient dark carved stone wall with an inset " +
  "panel of aged dark leather, subtle worn brass rivets, very low contrast and " +
  "even lighting so text stays readable when overlaid, dark fantasy palette, " +
  "digital painting in the style of Baldur's Gate 3, no text, no UI, no people, " +
  "no characters, no strong focal point, tileable-looking flat texture";

async function main() {
  const apiKey = process.env.OPENAI_API_KEY;
  if (!apiKey) {
    throw new Error('OPENAI_API_KEY not set in environment');
  }

  const res = await fetch('https://api.openai.com/v1/images/generations', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${apiKey}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      model: 'gpt-image-1',
      prompt: PROMPT,
      size: '1536x1024',
      quality: 'high',
    }),
  });

  if (!res.ok) {
    const text = await res.text();
    throw new Error(`HTTP ${res.status}: ${text}`);
  }

  const json = await res.json();
  const buf = Buffer.from(json.data[0].b64_json, 'base64');
  fs.mkdirSync(path.dirname(OUT_PATH), { recursive: true });
  fs.writeFileSync(OUT_PATH, buf);
  console.log(`[gen-ui-texture] wrote ${OUT_PATH} (${buf.length} bytes)`);
}

main().catch((err) => {
  console.error('[gen-ui-texture] failed:', err.message);
  process.exit(1);
});
```
